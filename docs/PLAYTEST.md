> Owned by the **Playtester** agent. Logs player-perspective observations, friction points, and suggestions. Design changes feed back to `[Game Designer]`; number changes feed back to `[Balancer]`.

## How to Use This Doc

Each session gets a dated entry. Use the template below. Keep observations factual ("the health bar was hard to see") rather than prescriptive ("make the health bar bigger") — the Playtester surfaces problems, the specialists solve them.
